Android Lab 5 - Campgrounds Updated

Submitted by: Jayden Scott

This is a app that has a list of national parks campgrounds that is able to still appear on user screen when turning on
airplane mode.

Time spent: 1 hour

Required Features

The core requirements for this project are similar to Unit 4, but this unit we will allow a user to see a list of campgrounds, with the campground name, description, location, and image photo while offline.

Most recently fetched data is stored locally in a database
If user turns on airplane mode and closes and reopens app, old data from the database should be loaded
